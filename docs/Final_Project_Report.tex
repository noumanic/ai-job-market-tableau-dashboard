\section{Project Overview}
This project serves as the final deliverable for the CS3012 Fundamentals of Data Visualization course. The primary objective is to analyze and visualize global AI, Machine Learning, and Data Science salary trends using real-world crowdsourced data. By leveraging a dataset of over 147,000 salary submissions, we have developed an interactive Tableau dashboard to answer critical industry questions regarding compensation, remote work dynamics, and regional hiring patterns.

\subsection{Core Research Questions}
The analysis focuses on five foundational questions:
\begin{itemize}
    \item Which AI/ML/Data Science roles command the highest salaries in the current market?
    \item How does compensation scale across different career levels (Entry Level to Executive)?
    \item What is the trajectory of remote work adoption from the post-pandemic peak in 2022 to the present?
    \item Which geographic regions lead in AI hiring and how do regional pay scales differ?
    \item How are salaries distributed across pay bands and quartiles within the industry?
\end{itemize}

\section{Dataset Description}
The project utilizes the "Global Salaries in AI, ML, Data Science and Big Data" dataset, sourced from \textit{aijobs.net} and \textit{foorilla.com}. 

\subsection{Source and Authenticity}
The dataset is published under a CC0 Public Domain license and is curated by the organizations that own the platforms. Unlike synthetic datasets, this data contains genuine indicators of a real tech labor market:
\begin{itemize}
    \item \textbf{High Skewness:} The salary distribution is right-skewed (skewness = 1.35), typical of professional earnings.
    \item \textbf{Geographic Concentration:} The US dominates with approximately 90\% of submissions.
    \item \textbf{Raw Variation:} Over 400 unique job title variants reflect real-world naming conventions.
\end{itemize}

\section{Preprocessing Pipeline}
A robust Python-based preprocessing pipeline was implemented to clean the raw data and engineer new features for enhanced storytelling in Tableau.

\subsection{Data Cleaning (\texttt{02\_clean.py})}
The cleaning stage focused on standardizing the population for a fair comparison:
\begin{enumerate}
    \item \textbf{Employment Type:} Restricted to Full-Time (FT) employees to remove noise from contract/freelance work.
    \item \textbf{Temporal Filter:} Kept records from 2022 to 2025.
    \item \textbf{Outlier Removal:} Removed records below the 1st percentile and above the 99th percentile (Bounds: \$37,974 to \$385,000).
\end{enumerate}

\subsection{Feature Engineering (\texttt{05\_feature\_engineering.py})}
To add depth to the visualization, 12 new columns were derived. These include salary bands, quartiles, and comparative metrics (relative to year/role averages).

\begin{lstlisting}[language=Python, caption=Excerpts from Feature Engineering Pipeline]
# 1. SALARY BAND - Categorical grouping for distribution analysis
bins   = [0, 60000, 100000, 150000, 200000, 260000, 385001]
labels = ["<$60K", "$60K-$100K", "$100K-$150K", "$150K-$200K", "$200K-$260K", "$260K+"]
df["salary_band"] = pd.cut(df["salary_in_usd"], bins=bins, labels=labels)

# 2. IS_US - Comparison between US and International markets
df["is_us"] = df["company_location"].apply(
    lambda x: "United States" if x == "US" else "Rest of World"
)

# 3. SALARY vs YEAR AVERAGE - Contextual measure for tooltips
year_avg = df.groupby("work_year")["salary_in_usd"].mean()
df["year_avg_salary"] = df["work_year"].map(year_avg).round(0)
df["salary_vs_year_avg"] = (df["salary_in_usd"] - df["year_avg_salary"]).round(0)
\end{lstlisting}

\section*{Appendix: Python Implementation}
\subsection*{02\_clean.py - Data Sanitization}
\begin{lstlisting}[language=Python]
import pandas as pd

# Load raw data
df = pd.read_csv("../data/raw/salaries.csv")

# 1. Keep Full-Time only
df = df[df["employment_type"] == "FT"]

# 2. Keep 2022 to 2025
df = df[df["work_year"] >= 2022]

# 3. Outlier removal (1st to 99th percentile)
lower = df["salary_in_usd"].quantile(0.01)
upper = df["salary_in_usd"].quantile(0.99)
df = df[(df["salary_in_usd"] >= lower) & (df["salary_in_usd"] <= upper)]

df.to_csv("../data/processed/salaries_clean_final.csv", index=False)
\end{lstlisting}
