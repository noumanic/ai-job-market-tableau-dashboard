\chapter*{Abstract}
This report provides an exhaustive analysis and visualization of the global Artificial Intelligence (AI) and Machine Learning (ML) job market from 2022 to 2025. Leveraging a real-world crowdsourced dataset of 147,348 cleaned records, the project employs a rigorous Python-based preprocessing pipeline to engineer 12 new analytical dimensions. We present 20 distinct visualizations that explore salary distributions, the "remote work collapse," geographic hiring patterns, and compensation growth across experience levels. The final deliverable is an interactive Tableau dashboard designed for executive-level decision-making.

\section{Project Context}
The AI job market is one of the most dynamic sectors of the global economy. As businesses pivot towards generative AI and LLMs, the demand for specialized talent has reached unprecedented levels. This project, titled "AI Job Market Intelligence," aims to provide a clear, data-driven narrative of this evolution.

\section{Objectives}
\begin{itemize}
    \item \textbf{Data Integrity:} Clean and normalize crowdsourced salary data.
    \item \textbf{Exploratory Analysis:} Identify key drivers of compensation.
    \item \textbf{Trend Analysis:} Document the shift in work modes and salary growth.
    \item \textbf{Dashboarding:} Create a professional-grade visualization in Tableau.
\end{itemize}

\chapter{Preprocessing Pipeline}

\section{Methodology}
The pipeline is designed for scalability and reproducibility. It consists of five distinct Python scripts that transform the raw 151,445 records into a refined 147,348-row dataset with 27 analytical columns.

\section{Pipeline Scripts}
\begin{enumerate}
    \item \texttt{01\_explore.py}: Initial EDA and profiling.
    \item \texttt{02\_clean.py}: Filtering and outlier removal.
    \item \texttt{03\_title\_grouping.py}: Mapping 406 job titles to 15 categories.
    \item \texttt{04\_summary\_stats.py}: Calculation of KPI metrics for the dashboard.
    \item \texttt{05\_feature\_engineering.py}: Engineering 12 new analytical columns.
\end{enumerate}

\section{Calculated Fields}
\begin{itemize}
    \item \textbf{Avg Salary USD:} \texttt{AVG([Salary In Usd])}
    \item \textbf{EX/EN Ratio:} \texttt{AVG(IF [Exp] = "EX" THEN [Sal] END) / AVG(IF [Exp] = "EN" THEN [Sal] END)}
\end{itemize}

\section{02\_clean.py}
\begin{lstlisting}[language=Python]
import pandas as pd
df = pd.read_csv("../data/raw/salaries.csv")
df = df[df["employment_type"] == "FT"]
df = df[df["work_year"] >= 2022]
q01, q99 = df["salary_in_usd"].quantile([0.01, 0.99])
df = df[(df["salary_in_usd"] >= q01) & (df["salary_in_usd"] <= q99)]
df.to_csv("../data/processed/salaries_clean.csv", index=False)
\end{lstlisting}

\section{03\_title\_grouping.py}
\begin{lstlisting}[language=Python]
def group_title(title: str) -> str:
    t = title.lower()
    if "scientist" in t: return "Data Scientist"
    if "machine learning" in t or "ml" in t: return "ML Engineer / MLOps"
    if "analyst" in t: return "Data Analyst"
    return "Other"
df["title_group"] = df["job_title"].apply(group_title)
\end{lstlisting}
